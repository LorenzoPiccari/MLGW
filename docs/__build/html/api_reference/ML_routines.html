class GDA(D, K, naive=True, hard_clustering=False, same_weights=False)[source]#

=== This class implements a model for Gaussian Discriminant Analysis. The model is a classifier with form:

p(y=k|x,params) ~ p(y=k) * p(x | y=k, params) = pi_k * N(x | mu_k, sigma_k)

__init__#

Initialize the model with K classes for regressions. Input:

D dimensionality of input space K number or classes for regression naive whether the covariance matrix for each class should be diagonal (naive assumption) hard_clustering whether hard clustering predictions should be done same_weights whethet all p(y=k) should be the same

Output:

class PCA_model(filename=None)[source]#

jac_extra_features(data, feature_list, log_list=None)[source]#

Features can be log preprocessed. Gradients are computed as follows:

where (xi_i)_j is the j-th augmented feature for the i-th data point Input:

Output:
